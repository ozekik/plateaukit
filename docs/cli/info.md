
# PLATEAU都市モデルの情報を取得 (CLI)

## 都市モデルの詳細情報を表示

インストール済みの都市モデルの詳細情報を表示するには、`plateaukit info` コマンドを実行します。
データセットに含まれているデータの種類や属性の種類が取得できます。

```bash title="例: 大阪府大阪市のデータの詳細情報を表示"
plateaukit info plateau-27100-osaka-shi-2022
```
